# 🍽️ Food Recognition and Calorie Estimation using Convolutional Neural Networks (CNN)

## 📌 Overview

This project implements a **Food Recognition and Calorie Estimation System** using **Convolutional Neural Networks (CNNs)**. The model classifies food images into their respective categories and estimates their calorie content, helping users make informed dietary choices.

The project demonstrates an end-to-end computer vision pipeline, from image preprocessing and model training to prediction and calorie estimation.

---

## 🎯 Problem Statement

Accurately identifying food items and estimating their nutritional value is a challenging task. This project aims to automate food recognition using deep learning and provide calorie estimates based on the predicted food category.

---

## 🚀 Features

- 🍕 Food image classification using CNN
- 📷 Image preprocessing and augmentation
- 🧠 Deep Learning-based feature extraction
- 🔍 Prediction on custom food images
- 🔥 Calorie estimation for recognized food items

## 🧠 Model Architecture

The project uses a **Convolutional Neural Network (CNN)** consisting of multiple convolutional and pooling layers followed by fully connected layers for image classification.

The pipeline includes:

- Image Preprocessing
- Data Augmentation
- CNN Training
- Food Classification
- Calorie Estimation
